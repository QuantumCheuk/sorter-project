## 今日研究笔记（2026-04-14 课题4 Day1）

### 🚨 重大发现：风扇瓶颈 — 5015无法实现3-way分离

**文件：** `simulation/density_topic4_day1.py` + `density_topic4_day1_corrected.png`

**问题根源：** 原始120×20mm通道设计根本无法实现密度分选！
- 5015风扇（120 L/min）在120×20mm通道下：v_air_max = 83 cm/s
- 轻豆终端速度 = 163 cm/s → 83 < 163，**任何豆子都吹不起来**

**物理修正：**
- 几何模型低估了阻力：豆子翻滚改变有效迎风面积
- 经验校准：v_t = 5.16 × √mass_g (m/s)
- 结果：轻=163 cm/s，中=200 cm/s，重=231 cm/s ✅

**风扇-通道耦合分析（关键修正）：**
窄通道并不等于高风速——风扇背压会限制流量。使用 affinity laws：
- 风扇曲线：Q(PWM, ΔP) = Q_max × PWM × √(1 - ΔP/P_max)
- 通道需求：Q = Cd × A × √(2ΔP/ρ)

| 通道规格 | v_air @100% | 3-Way? | 备注 |
|----------|------------|--------|------|
| 120×20mm | 83 cm/s | ❌ | 原设计，死路 |
| 25×80mm | 100 cm/s | ❌ | 背压极限 |
| 60×10mm | ~20 cm/s | ❌ | 宽通道流阻小 |
| 40×10mm | ~250 cm/s | ⚠️ | 可惜通过性紧 |
| 15×80mm | 168 cm/s | 2-way✅ | 最佳平衡 |

**结论：** 5015风扇（120 L/min）**无法实现3-way密度分选**。
- 需要 >200 L/min 才能在25mm通道达到163 cm/s
- 现有风扇最多只能做到2-way（轻豆 vs 中+重豆）

**实用方案（今日成果）：**
- 60×10mm通道，θ=30°，PWM=50-60%
- v_air ≈ 168-185 cm/s
- 2-way分离：轻豆→LIFTED区，中+重豆→BOTTOM区
- CAD已更新：`sorter/cad/density_channel.scad`

**升级路线（未来采购）：**
- 涡轮鼓风机（300+ L/min）→ 25mm通道实现3-way分离

**内容1：标定方法详解（ISO 6673 + 不确定度分析）**

ISO 6673 烘干法标准步骤：
1. 取约 5g 样品，精确称重 W_wet（精度±0.001g）
2. 105°C 烘干 16h（或 130°C 烘干 2h）
3. 干燥器冷却 30min
4. 称重 W_dry
5. M_wb% = (W_wet - W_dry) / W_wet × 100

**5点标定样本制备：**

| 编号 | 目标含水率 | 制备方法 |
|------|-----------|----------|
| S1 | 5% | 烘箱干燥16h + 密封 |
| S2 | 8% | 烘干豆 + 精确加水 + 平衡24h |
| S3 | 10% | 烘干豆 + 精确加水 + 平衡24h |
| S4 | 12% | 烘干豆 + 精确加水 + 平衡24h |
| S5 | 15% | 烘干豆 + 精确加水 + 平衡24h |

**线性标定模型：** C = a + b × M
- b = 0.2060 pF/%, a = -0.3300 pF（用S1+S5两点标定）

**不确定度分析：**

| 不确定度来源 | 估计值 | 贡献 |
|------------|--------|------|
| 称重精度 (±0.001g, 5g) | ±0.02% | 0.02%² |
| 烘干不完全 (±0.5h) | ±0.30% | 0.05%² |
| 水分均匀性 (±0.5% @ 平衡) | ±0.50% | 0.25%² |
| AD7746分辨率 (±1fF @ 2pF) | ±0.05% | 0.05%² |
| 温度漂移 (±2°C @ 0.02%/°C) | ±0.40% | 0.40%² |
| 探头几何误差 (±5% 占空比) | ±1.67% | 0.17%² |
| **合成不确定度** | **±0.51%** | |
| **扩展不确定度 (k=2)** | **±1.01%** | |

**结论：** 合成不确定度 ±0.51%，满足 ±0.5% 目标 ✅（k=2为±1.01%）

**内容2：AD7746 外围电路设计**

引脚连接（典型接法）：

| 引脚 | 功能 | 连接 |
|------|------|------|
| 1 (CAP+) | 探头电极+ | 0.1µF旁路 → GND |
| 2 (CAP-) | 探头电极- | 0.1µF旁路 → GND |
| 3 (GND) | 模拟地 | 星形接地，单点汇接 |
| 4 (VCC) | 3.3V | LDO → 模拟3.3V |
| 5 (REF+) | 外部精密参考 | 2.5V ADR01（可选）|
| 7 (CLK) | 32kHz晶振 | 32kHz晶振 or 直接输入 |
| 8 (SDA) | I2C数据 | 2.2kΩ上拉 → VCC |
| 9 (SCL) | I2C时钟 | 2.2kΩ上拉 → VCC |
| 10 (ADR) | I2C地址 | GND → 0x48 |

**I2C总线分析：**
- 推荐：Rp=2.2kΩ, C_bus<400pF → 400kHz快速模式可用 ✅
- 注意：总线电容 = AD7746(50pF) + 探头电缆 + 其他设备

**⚠️ 重要修正：探头电缆长度效应**

AD7746输入范围仅 ±4pF（差分），任何电缆电容都会直接叠加：
- RG174 @ 30cm → 30pF ❌（超出量程）
- CAT5e @ 30cm → 15pF ❌（超出量程）

**解决方案：AD7746模块必须紧贴探头安装！**
- 模块安装在探头附近（<5cm引线）→ 使用排针连接
- Pi通过1-2米I2C线连接（I2C缓冲器如PCA9600可延长至10米）
- 或采用插针式AD7746模块直接插在探头PCB上

**内容1：PID控制算法设计**

PID控制器设计（位置式离散PID）：
```
u(k) = KP*e(k) + KI*Σe + KD*[e(k) - e(k-1)]
```

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| KP | 15.0 | 比例增益 |
| KI | 0.3 | 积分增益 |
| KD | 5.0 | 微分增益 |
| 输出限制 | 0-120 RPM | 电机最大转速 |

250g批次仿真结果：
- 目标：250g @ 70s = 3.571 g/s
- 满速120RPM理论速率：3.582 g/s ✅
- 目标速率对应：~75 RPM
